Existem dois principais tipos de dados: rotulados e não rotulados

Imagine que você tem um dataset com 10.000 fotos de vários animais, antes de qualquer modelagem alguém revisa as fotos selecionado apenas as que são que cachorros, e isso cria um conjunto de dados estruturados (labelled dataset). E isso pode se extender pra qualquer tipo de dados.

Qual a vantagem de construir uma modelagem em cima de dados rotulados? Você pode treinar um modelo que irá se aperfeiçoando com o passar do treino. Modelos treinados com dados rotulados são mais confiáveis e tem bom desempenho em modo real.

A outra opção são os dados não rotulados, hoje em dia os modelos de machine learning um pouco mais avançados, nos permitem trabalhar com dados não rotulados. Isso significa alimentar o conjunto de dados de 10.000 fotos antes de passar por cada uma delas.
