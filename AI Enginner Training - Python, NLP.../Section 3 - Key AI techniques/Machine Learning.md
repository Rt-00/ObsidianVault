Subcampo fundamental da IA

A Ideia do machine learning se resume a projetar um sistema capaz de aprender e se melhorar por meio de um processo de tentativa e erro.

Considerando um modelo de aprendizado de máquina semelhante a um aluno ensinado por um professor. Nesse caso o modelo de Machine Learning é o aluno e o cientista de dados é o professor. O papel do professor é ajudar o aluno a aprender a resolver problemas. Para fazer isso, o professor deve fornecer ao aluno materiais de aprendizado adequados (dados). Com informações suficientes o aluno estuda e absorve os materiais de treinamento. Da mesma forma o algorítimo de machine learning aprende a reconhecer os padrões nos dados de treinamento alimentados.

O objetivo é preparar os alunos para serem bem sucedidos no exame final, com perguntas desconhecidas que testam o material abordado. No caso de ML o objetivo é ensinar a resolver um problema específico com dados que ele nunca viu. Quanto mais atualizados e melhores os dados, melhor chance de resolver um problema que nunca viu. 

Um modelo simples com muitos dados bons pode se sair melhor do que um modelo complicado com menos dados.

A adoção de novos métodos de ensino e adaptação dos estilos aos alunos podem melhorar significativamente o desempenho deles.

Exemplo de caso de uso em uma empresa:
Um agente imobiliário planeja desenvolver um aplicativo mobile para fornecer aos clientes em potencial uma estimativa do preço de vendas de suas casas. Os usuários podem receber essa estimativa respondendo a perguntas específicas no aplicativo. Isso pode impulsionar significativamente os negócios do agente imobiliário, permitindo que ele colete detalhes de contato de pessoas interessadas em vender suas propriedades. Para colocar o plano em ação. o agente imobiliário entrou em contato com um cientista de dados e perguntou se o projeto era viável. A primeira pergunta do cientista de dados foi sobre a disponibilidade de uma lista suficientemente grande de transações anteriores. Felizmente, a empresa do corretor de imóveis tinha acesso a um bando de dados com milhares de transações passadas.

Então o Cientista de Dados partiu para a criação do modelo.

Nesse caso:

$F(x) = y$
`y = futuros preços das casas`
`x = characteristics das propriedades`

Imagine que queremos desenvolver um modelo de aprendizado de máquina para prever os preços dos imóveis em nossa cidade, analisando transações imobiliárias anteriores em seu conjunto de dados de treinamento. Portanto temos muitas transações anteriores em que X e Y são conhecidos. Então devemos criar um modelo que interprete as características das propriedades (x) e preveja o preço (y) com base nas transações anteriores. Os usuários adicionam as características de suas moradias e com base no histórico de transação e nas características da propriedade prevemos o preço.
