Supervised Learning é excelente quando trabalhamos com dados rotulados. No Supervised Learning treinamos o modelo explicitamente com resultados conhecidos para orienta-lo. Bom para resolver problemas de classificação como o problema das 10.000 fotos e tem que identificar o cachorro, ou problemas de regressão que com base em dados anteriores prevê algum resultado.

Unsupervised Learning é interessante para categorizar e rotular dados. No exemplo das 10.000 fotos ele pegaria fotos de todos os grupos de animais e separaria em conjuntos de acordo com suas características, mas não identificaria cada um dos conjuntos apenas agruparia por padrões. Ou poderia utilizar um modelo de Unsupervised Learning para determinar quais tipos de propriedades são vendidos com mais frequências.

Reinforcement Learning é utilizado junto com dados não rotulados e aplicado em cenários  específicos em que a máquina descobre como atingir uma meta designada. Combina com o ML Supervisionado a compreensão do resultado desejado, basicamente, direcionamos o computador para otimizar o cumprimento de nossas metas. Diferentemente do ML Supervisionado não fornecemos dados rotulados ao modelo, em vez disso criamos regras. O modelo de ML aprende por tentativa e erro dentro de parâmetros específicos de nossas regras criadas. Muito utilizado na robótica e nos sistemas de recomendação online.
